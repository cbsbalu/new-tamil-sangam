Overview

This file describes the Northeast Wisconsin Tamil Sangam's wesbite structure and includes step-by-step instructions to update the website content. Familiarity with basic HTML will be useful to update the text and image content of the website. Knowledge of HTML, JavaScript and CSS is required to update the design and/or add new pages and functionalities to the website.

Files & Folders

File/FolderDescription
index.htmlEnglish page
index_ta.htmlTamil page
scripts.jsJava Script file
styles.cssStylesheet file
images/Folder contains logo and other site images
gallery/Folder contains the photo gallery images

This website is a single page website with index.html containing English content and index_ta.html containing Tamil content. Both files are identical except for the loaded fonts and content language. They contain the following 4 sections:

IDTitle
homeHome
directoryDirectory
memberMembership
contactContact Us

Each of the above sections content are coded inside a DIV tag (<div id="section-id">.....</div>). When the page loads, home section is shown and others are hidden. When visitor clicks a menu link, the corresponding section's DIV is shown and all other section DIV's are hidden. The JavaScript function navigateTo(section-id) performs this show/hide operations. The HTML code of the menu link is in the format: <a href="#" onclick="navigateTo('section-id')">Section Title</a>.

Fonts and Styles

The website uses Google Fonts, Font Awesome CSS styles, Style elements from styles.css and inline Style elements.

HTML tags

The website sections uses standard HTML tags to display text, links and images.

  • <h1> to <h5> for heading text.
  • <p> for paragraphs.
  • <ol>/<ul> and <li> for ordered/unordered lists and list items.
  • <a href="#" onclick="navigateTo(section-id);">...</a> for hyperlinks to internal sections of the website.
  • <a href="https://www.abc.com">...</a> for hyperlinks to external websites.
  • <img src="..." width=".." style="..." /> with appropriate width, height and style parameters for images.

Announcement

A site wide Announcement can be displayed at the right area of the website. Announcement can include Heading and Description text and will be visible in the right area of the website on all pages when viewed in a Desktop/Tablet and on the bottom area when viewed in a Mobile device.

Photo Gallery

The Photo Gallery in the Home section contains an image gallery feature. This section uses Java Script function loadGallery() from scripts.js and CSS elements from styles.css. The script loads thumbnails of images from the gallery/ folder and when the thumbnails are clicked, the corresonding image is enlarged and displayed. The loadGallery() function is called when the page loads using the body onload event.

How to update Announcement

The following are the instructions to include a new announcement:

  1. Take a backup of index.html and index_ta.html.
  2. Open index.html in a text editor and go to the DIV section with the following code: <div class="announcement">. Delete the content inside this DIV area (untill the closing </div>) and include your new Heading and Paragraph text. Use HTML tags <h2> for heading text and <p> for paragraph text. Save the file.
  3. Open index_ta.html and repeat the above step to include the Tamil translation (You can use Google Translate to translate English content to Tamil). in the corresponding area of the file. Save the file.
  4. Preview index.html & index_ta.html pages in a browser and verify that the announcement appears fine.

How to update Photo Gallery

The Photo Gallery contains an image gallery feature. Refer to Photo Gallery in the Overview topic in this page to know more about this feature. The following are the instructions to update the Photo Gallery with new set of images.

  1. Delete the current images in the gallery/ folder. Take a backup of index.html, index_ta.html and scripts.js files.
  2. Upload new images to gallery/ folder. Name the images as 1.jpeg, 2.jpeg etc..
  3. Update scripts.js file totalImages variable in the loadGallery() function with the total number of images in the gallery/ folder. If you are using an image format other than .jpeg, update the fileFormat variable with the file extension you are using.
  4. Update the index.html and index_ta.html Photo Gallery section with appropriate Heading and Paragraph about the program.
  5. Preview the index.html & index_ta.html in a browser and go to Photo Gallery section and verify that the new images appear fine in both Thumbnail and enlarged view.

How to update Photo Album/Video link & Thumbnail image

After a Tamil Sangam organized event, the Past Events in Home section can be updated with a link to a newly created Photo Album/Video. In the table of Thumbnails, you will be doing the following:

  • Insert a new Column at the beginning (This column will contain the code block of thumbnail image and link to the new Album/Video).
  • Delete the last Column (This will be an old event that can be removed).
  • Move a Column to next row (This is to adjust the table rows appropriately).
Instructions:
  1. Login to Google Photos using the Sangam's email account.
  2. Create a new Album and upload the Photos.
  3. Click the "Share" button from the album and copy the Address of the Album (to be used as ALBUM ADDRESS).
  4. Click on an individual photo you wish to use as Thumbnail and copy the Address (to be used as THUMBNAIL ADDRESS). Note: If this is a YouTube video you are linking, create a Thumbnail image using Microsoft Paint and upload to /images folder and use the same for THUMBNAIL ADDRESS.
  5. Construct a code block as below:
     <div class="column">
        <a href="ALBUM ADDRESS" target="_blank">
         <img src="THUMBNAIL ADDRESS" class="home-thumb" alt="EVENT DESCRIPTION"/>
         EVENT DESCRIPTION
       </a>
     </div>
  6. Take a back up of index.html and index_ta.html.
  7. Open the index.html in a text editor and look for comments: <!--Photo Album/Video link & Thumbnail image-->.
  8. Insert the new code block you have created below <div class="row">.
  9. Delete the last Column code block of the table.
  10. Move one Column code block from first row to the second row of the table.
  11. Save index.html.
  12. Repeat the same in index_ta.html with Tamil translation for the event description.
  13. Preview index.html and index_ta.html.

How to update section content

Each section content are present inside a DIV tag with the id of the section. For example, if you want to update the Home section content, look for <div id="home" and update the content. Refer to the Overview topic in this page to know more about the sections. The following are the instructions to update a section with new content.

  1. Take a backup of index.html and index_ta.html.
  2. Open index.html in a text editor and go to the DIV section you want to update and make changes within the sections DIV tag area. Save the file.
  3. Open index_ta.html in a text editor and include the Tamil translation (You can use Google Translate to translate English content to Tamil) in the corresponding DIV tag area. Save the file.
  4. Preview index.html & index_ta.html pages in a browser and verify that the content appears fine.

To include a hyperlink in your content, use A HREF tag with a call to navigateTo function.

  • Example: <a href="#" onclick="navigateTo('section-id');">Section Title</a>

To include an individual image in your content, upload the image to the images/ folder and use IMG tag to display the image. Use width, height & style element parameters to display/align the image appropriately.

  • Example: <img src="images/1.jpeg" width="50%" style="float: right; padding-top: 25px; padding-bottom: 25px; padding-left: 25px;"/>
