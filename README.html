Overview

This file describes the Northeast Wisconsin Tamil Sangam's wesbite structure and includes step-by-step instructions to update the website content. Familiarity with basic HTML will be useful but not required to update the text and image content of the website. Knowledge of HTML, JavaScript and CSS is required to update the design and/or add new pages and functionalities to the website.

This website is a single page website with index.html containing English content and index_ta.html containing Tamil content. Both files are identical except for the loaded fonts and content language. They contain the following 6 sections:

IDTitle
homeHome
eventsEvents
galleryGallery
directoryDirectory
memberMembership
contactContact Us

Each of the above sections content are inside a DIV tag (<div id="section-id">.....</div>). When the page loads, home section is shown and others are hidden. When visitor clicks a menu link, the corresponding section's DIV is shown and all other section DIV's are hidden. The JavaScript function navigateTo(section-id) performs this show/hide operations. The HTML code of the menu link is in the format: <a href="#" onclick="navigateTo('section-id')">Section Title</a>.

Image Gallery

The Gallery section contains an image gallery feature. This section uses Java Script function loadGallery() from scripts.js and CSS elements from styles.css. The script loads thumbnails of images from the gallery/ folder and when the thumbnails are clicked, the corresonding image is displayed in larger size. The loadGallery() function is called when the page loads using the body onload event.

HTML tags

The website sections uses standard HTML tags to display text, links and images.

  • <h1> to <h5> for heading text
  • <p> for paragraphs
  • <ol>/<ul> and <li> for ordered/unordered lists and list items
  • <a href="#" onclick="navigateTo(section-id);">...</a> for hyperlinks to internal sections of the website
  • <a href="https://www.abc.com">...</a> for hyperlinks to external websites
  • <img src="..." width=".." style="..." /> with appropriate width, height and style parameters for images.

Fonts and Styles

The website uses Google Fonts, Font Awesome CSS styles, Style elements from styles.css and inline Style elements.

How to update Gallery

The Gallery section contains an image gallery feature. Refer to Image Gallery in the Overview topic in this page to know more about this feature. The following are the instructions to update the Gallery with new set of images.

  1. Delete the current images in the gallery/ folder. Take a backup of index.html, index_ta.html and scripts.js files.
  2. Upload new images to gallery/ folder. Name the images as 1.jpeg, 2.jpeg etc..
  3. Update scripts.js file loadGallery() function totalImages variable with the total number of images in the gallery/ folder. If you are using an image format other than .jpeg, update the loadGallery() function fileFormat variable with the file extension you are using.
  4. Update the index.html and index_ta.html Gallery section with appropriate Heading and Paragraph about the program.
  5. Preview the index.html & index_ta.html in a browser and go to Gallery section and verify that the new images appear fine.

How to update section content

Each section content are present inside a DIV tag with the id of the section. For example, if you want to update the Home section content, look for <div id="home" and update the content. Refer to the Overview topic in this page to know more about the sections. The following are the instructions to update a section with new content.

  1. Take a backup of index.html and index_ta.html.
  2. Open index.html in a text editor and go to the DIV section you want to update and make changes within the sections DIV tag area. Save the file.
  3. Open index_ta.html in a text editor and include the Tamil translation (You can use Google Translate to translate English content to Tamil) in the corresponding DIV tag area. Save the file.
  4. Preview index.html & index_ta.html pages in a browser and verify that the content appears fine.

To include a hyperlink in your content, use A HREF tag with a call to navigateTo function.

  • Example: <a href="#" onclick="navigateTo('section-id');">Section Title</a>

To include an individual image in your content, upload the image to the images/ folder and use IMG tag to display the image. Use width, height & style element parameters to display/align the image appropriately.

  • Example: <img src="images/1.jpeg" width="50%" style="float: right; padding-top: 25px; padding-bottom: 25px; padding-left: 25px;"/>
